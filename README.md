# AI Resume Analyzer & Job Recommendation System

This project analyzes resumes using NLP techniques to extract skills and recommend suitable job roles. It also highlights missing skills to help users improve employability.

## Features
- Resume PDF upload
- Skill extraction using NLP
- Job role recommendation
- Missing skill analysis

## Tech Stack
- Python
- Streamlit
- NLP
- Machine Learning

## How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
